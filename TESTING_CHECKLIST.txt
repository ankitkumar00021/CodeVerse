## 🚨 IF YOU ENCOUNTER ISSUES

### Blank White Screen
```
1. F5 to refresh
2. Ctrl+Shift+Delete to clear cache
3. Check F12 console for errors
4. Restart both servers
```

### MongoDB Connection Error
```
1. Check DB_URL in backend/.env
2. Test connection string format
3. Verify MongoDB Atlas is running
4. Check IP whitelist in MongoDB Atlas
```

### CORS Error in Console
```
1. Backend must be on http://localhost:3000
2. Frontend must be on http://localhost:5173
3. Check CORS config in backend/src/server.js
```

### Code Execution Fails
```
1. Check API response in Network tab (F12)
2. Verify Piston API keys in env
3. Check backend logs
4. Try simple code first
```
